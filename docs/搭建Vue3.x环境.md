# 搭建Vue3.x环境

## 一、通过Vue-cli创建我们的项目

### **1、安装Vue-cli，在同一个电脑上面只需要安装一次**

```shell
npm install -g @vue/cli
```

或者使用淘宝镜像

```shell
cnpm install -g @vue/cli
```

如果电脑上面没有安装过cnpm可以通过下面命令安装：

```shell
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

### **2、通过Vue-cli创建项目**

创建名为hello-vue3的项目

```shell
vue create hello-vue3
```
